Guide de l'API Forfaits

#1 - Affichage de la liste des forfaits en format JSON dans le navigateur Chrome

  1. Télécharger les fichiers du projet soumis sur google classroom localement sur votre ordinateur.
  2. Ouvrir le site api-forfaits dans chrome à partir du "Localhost" dans Ampps.
  3. Dans la barre d'url ajouter "liste-forfaits-json.php" à la fin de l'adresse.

#2 - Affichage la fiche du forfait par leur ID en format JSON dans le navigateur Chrome

  1. Dans la barre d'url, modifier le "liste-forfaits-json.php" par "fiche-forfaits-json.php/?id=1".
  2. Répétez l'étape #1 en modifiant la valeur de l'id par 2, 3, 4, ou 5 pour consulter la fiche JSON des autres forfaits existants dans la base de données.

#3 - Ajouter un forfait avec des données valides par Thunder Client

  1. Importer la base de données api-forfaits dans Ampps.
  2. Ouvrir le dossier api-forfaits dans VS Code.
  3. Aller dans l'extension Thunder Client et importer la collection API-FORFAIT
  4. Sélectionnez la requête "POST FORFAIT".
  5. Les données requises sont déjà saisies.
  6. Appuyer sur le bouton "Send" dans la barre de la requête.
  7. La requête est exécutée et nous avons la confirmation de l'ajout dans la section "Response" à droite.
  8. Vérifier l'ID du nouveau forfait en faisant la requête "GET FORFAITS" et notez-le.

#4 - Ajouter un forfait avec des données invalides par Thunder Client

  1. Sélectionner la requête "POST FORFAIT".
  2. Effacer la donnée "name" de les paramètres.
  3. Les données sont déjà saisies pour arriver à une erreur.
  4. Appuyer sur le bouton "Send" dans la barre de la requête.
  5. La requête est exécutée et nous avons l'erreur dans la section "Response" à droite.

#5 - Modifier un forfait avec des données valides par Thunder Client

  1. Sélectionner la requête "PUT FORFAIT".
  2. Indiquer l'ID dans la barre de la requête.
  3. Appliquer une modification dans les champs "regular_price" à 499.99 et "promotion_price" à 299.99
  4. Appuyer sur le bouton "Send" dans la barre de la requête.
  5. La requête est exécutée et nous avons la confirmation de la modification dans la section "Response" à droite.

#6 - Modifier un forfait avec des données valides sans identifiant par Thunder Client

  1. Reprendre la requête "PUT FORFAIT".
  2. Effacer l'ID dans la barre de la requête.
  3. Modifier le code du forfait pour : TEST NAME SANS ID.
  4. Appuyer sur le bouton "Send" dans la barre de la requête.
  5. La requête est exécutée et nous avons une erreur dans la section "Response" à droite.

#7 - Modifier un forfait avec des données invalides par Thunder Client

  1. Sélectionner la requête "PUT FORFAIT".
  2. Indiquer l'ID dans la barre de la requête.
  3. Effacer la donnée "name" de les paramètres.
  4. Appuyer sur le bouton "Send" dans la barre de la requête.
  5. La requête est exécutée et nous avons une erreur dans la section "Response" à droite.

#8 - Supprimer un forfait avec des données valides par Thunder Client

  1. Sélectionner la requête "DELETE FORFAIT".
  2. Indiquer l'ID dans la barre de la requête
  3. Appuyer sur le bouton "Send" dans la barre de la requête.
  4. La requête est exécutée et la confirmation de la suppression dans la section "Response" à droite.

#9 - Supprimer un forfait avec des données valides sans indentifiant par Thunder Client

  1. Reprendre la requête "DELETE FORFAIT".
  2. Effacer l'ID dans la barre de la requête.
  3. Appuyer sur le bouton "Send" dans la barre de la requête.
  4. La requête est exécutée et nous avons une erreur dans la section "Response" à droite.

#10 - Afficher la liste des forfaits après les opérations faites des points 2 à 8 par Thunder Client

  1. Sélectionner la requête "GET FORFAITS"
  2. Appuyer sur le bouton "Send" dans la barre de la requête.
  3. La requête est exécutée et les données des forfaits sont affichées dans la section "Response" à droite.

#11 - Afficher la liste des attractions par Thunder Client

  1. Sélectionner la requête "GET ATTRACTIONS"
  2. Appuyer sur le bouton "Send" dans la barre de la requête.
  3. La requête est exécutée et les données des attractions sont affichées dans la section "Response" à droite.
